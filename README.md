Exercices HTML/CSS – TP1 et TP2
Auteur :** Hanin Hamdi  
Cours :** Programmation Web  

Description
Ce dépôt contient les deux premiers travaux pratiques du cours de programmation web.  
Ils couvrent les notions de base de HTML5 et CSS : structure de page, mise en forme, sélecteurs, et mise en page 
## Contenu
- `tp1/entreprise.html` — Page HTML principale avec code css du TP1   
- `tp2/tp.html` — Deuxième exercice avec mise en page avancée  
- `tp2/tp.css` — Styles associés
